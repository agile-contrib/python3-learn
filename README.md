## How to use?

由于我们的代码是基于python3，首先，需要查看版本：

```
python --version
```

如果没有安装，可以去[官网](https://www.python.org/)进行下载。

准备好了以后，执行下面命令即可运行：

```
cd src
python xxx.py
```

其中，xxx.py表示你需要运行的文件。
